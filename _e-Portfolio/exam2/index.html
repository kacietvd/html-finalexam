advancement.

Where did it all begin?

"According to many people, the first computer programmer was the English
noblewoman Ada Lovelace. In 1843, she published a sequence of steps to
perform using a computing machine designed by her friend, Charles
Babbage. These notes are considered the first computer program."
"The program that Ada Lovelace wrote was for Charles Babbage's "Analytical
Engine", a mechanical general-purpose computer he designed, but never
completed. The design included memory, an arithmetic logic unit
(processor), and control flow for looping. This means it would have had
all of the key elements of a modern computer."
Computer Programming - History of Computer Programming. (n.d.). Retrieved November 14, 2020, from
https://www.softschools.com

The First High Level Programming Language

Created in 1957 by John Backus, Fortran (short for Formula Translation) is possibly
the oldest programming language that’s still in use today and is the first high
level programming langage invented. It’s designed to do complex statistical,
mathematical, and scientific work.
